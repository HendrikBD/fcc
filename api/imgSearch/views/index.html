Image Search API

An api that searches for a given image and returns the top results including the image URL, page URL and alt text.

Example Query:

https://img-search-bhd.glitch.me/img?q=cat&offset=3

Will return:

[{alt_text: "paws", img_url: "http://cat-breeds-enc...", page_url: "http://cat-breeds-enc..."}, ...]

Query Recent Queries:

https://img-search-bhd.glitch.me/recent
