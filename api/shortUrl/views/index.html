URL Shortener Microservice

Short-URL Creation:

https://url-bhd.glitch.me/new/https://google.ca
https://url-bhd.glitch.me/new/https://foo.com:3000

Short-URL Output:

{ "original_url":"http://google.ca", "short_url":"https://url-bhd.glitch.me/3XY" }
{ "original_url":"http://foo.com:3000", "short_url":"https://url-bhd.glitch.me/A4R" }

Short-URL Usage:

https://url-bhd.glitch.me/3XY"
https://url-bhd.glitch.me/A4R"
